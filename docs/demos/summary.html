You can nest directories as this!

toc.ron in this directory is:

(
    summary: ("Demos", "summary.adoc"),
    items: [
        File("Demo", "demo.adoc"),
        File("Demo (diagrams)", "demo_diagrams.adoc"),
    ],
)
Titles can be skipped as this: File("Demo", "demo.adoc"). Then it is automatically extracted from the AsciiDoc file.
