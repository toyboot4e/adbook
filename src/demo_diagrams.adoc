= Asciidoctor-diagrams demo
:revdate: Oct 19, 2020
:author: toyboot4e

Generate diagrams from ascii text!

== Requirements

Make sure you have asciidoctor-diagram:

[source,sh]
----
$ rvm install asciidoctor-diagram
----
